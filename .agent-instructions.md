# Agent Instructions - Robert's Portfolio

When starting any new feature, redesign, major refactoring, or structural modification in this workspace, the agent **MUST** always execute the `/grill-me` protocol.

## Protocol Rules:
1. **Interview First:** Do not write source code or begin implementation plans without first aligning on requirements.
2. **One Question at a Time:** Run the `/grill-me` interview, asking questions one by one using the interactive `ask_question` tool.
3. **Provide Recommendations:** For each design branch or feature trade-off, offer a recommended option clearly as the first choice.
4. **Research Codebase:** If a question can be answered by looking at the existing codebase, do so first instead of asking.
